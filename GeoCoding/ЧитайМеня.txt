Столбцы файла: «GlobalId», «Address», дальше не важно, регистр не важен. Разделитель ";"
Кодировка входных данных должна быть UTF-8, выходных будет UTF-8

1) Кнопка «Выбрать файл с данными» (можно просто перетащить файл на окно программы)
2) Если стоит галочка «Данные получить сразу», таблица данных заполнится данными из файла.
Имя файла для сохранения заполнится по шаблону: «Дата_Имя файла_UpLoad_Количество записей в файле»
3) Данный также можно получить из базы данных. Открыть вкладку "БД" и нажать кнопку "Свой запрос"
4) Если нужно поменять имя, кнопка: «Выбрать файл для сохранения»
5) Если нужно итоговый файл разбить на части (по строкам), ставим галочку и указываем нужное количество строк (итоговое число строк в файле = количество строк + строка заголовок)
Если итоговые данные разбиты по файлам, имена файлов: «Имя файла_Номер части»
6) Выбираем действия после геокодирования: «Сохранить результат», «Открыть папку результатов после сохранения»
7) Выбираем нужный геокодер (Тест не надо, это просто рандом)
8) Жмем кнопку: «Геокодировать». Ждем
9) Если стоит галочка: «Сохранить результат», то все готово. Если нет, кнопку: «Сохранить данные»
10) На вкладке ошибки, можно посмотреть адреса с ошибками. Если есть варианты, можно выбрать нудный вариант геокодирования
