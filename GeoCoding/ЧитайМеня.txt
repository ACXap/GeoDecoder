Столбцы файла: «GlobalId», «Address», дальше не важно, регистр не важен.

1)	Кнопка «Выбрать файл с данными» (можно просто перетащить файл на окно программы)
2)	Если стоит галочка «Данные получить сразу», таблица данных заполнится данными из файла.
Имя файла для сохранения заполнится по шаблону: «Дата_Имя файла_UpLoad_Количество записей в файле»
3)	Если нужно поменять имя, кнопка: «Выбрать файл для сохранения»
4)	Если нужно итоговый файл разбить на части (по строкам), ставим галочку и указываем нужное количество строк (итоговое число строк в файле = количество строк + строка заголовок)
Если итоговые данные разбиты по файлам, имена файлов: «Имя файла_Номер части»
5)	Выбираем действия после геокодирования: «Сохранить результат», «Открыть папку результатов после сохранения»
6)	Жмем кнопку: «Геокодировать». Ждем
7)	Если стоит галочка: «Сохранить результат», то все готово. Если нет, кнопку: «Сохранить данные»
